What is <colgroup>?

<colgroup> is used in an HTML table to group one or more columns so you can apply styles or attributes to those columns.

Instead of styling each cell individually, you style the entire column.

Basic Structure

<table>
  <colgroup>
    <col>
    <col>
  </colgroup>

  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>

  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>
