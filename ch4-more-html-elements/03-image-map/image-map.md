What is an image Map? 

An image map lets uou make different parts of an image clickable. 

For example:
In the index.html if you click part of the cat's face it opends another page. 

If you click the cats body it opens another page. 

Remember!
Usermap contacts the image to the map.
This is done by using usemap and name attribute to connect.

e.g

usemap="#cutecatmap" → tells the image which map to use

name="cutecatmap" → identifies the map

So the browser connects them.

The # just means reference to that name.

Simple notes version:

usemap → used in <img> to connect to a map
name → used in <map> to identify the map

Rule:

usemap="#name"
map name="name"
