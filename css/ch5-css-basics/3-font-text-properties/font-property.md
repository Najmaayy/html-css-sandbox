font-size: 30px;

Controls the size of the text.

font-size: 30px;

30px means the text height is 30 pixels.

Larger number → larger text.

Example effect:

Small text → Big text
font-weight: 700;

Controls how thick or bold the text is.

font-weight: 700;

Common values:

Value	Meaning
400	normal
700	bold

So 700 means bold text.

Example:

Normal → Bold
font-style: normal;

Controls the style of the text.

font-style: normal;

Possible values:

normal → regular text

italic → slanted text

oblique → slightly slanted text

Example:

normal → italic
font-variant: small-caps;

Displays lowercase letters as smaller capital letters.

font-variant: small-caps;

Example idea:

hello world

becomes

HELLO WORLD

But original capital letters remain slightly larger.

text-transform: capitalize;

Changes the capitalization of text.

text-transform: capitalize;

Each word starts with a capital letter.

Example:

hello world

becomes

Hello World

Other values include:

uppercase

lowercase

text-indent: 40px;

Adds space at the beginning of the first line of text.

text-indent: 40px;

Used often in paragraph formatting.

Example concept:

     This paragraph starts with space.
letter-spacing: 2px;

Controls the space between letters.

letter-spacing: 2px;

Example:

HELLO

becomes

H E L L O

Letters are 2 pixels further apart.

word-spacing: 150px;

Controls the space between words.

word-spacing: 150px;

Example:

Hello world today

becomes something like:

Hello        world        today

150px is very large spacing.

text-align: center;

Controls horizontal alignment of text.

text-align: center;

Possible values:

Value	Meaning
left	align left
right	align right
center	center text
justify	spread text evenly

Example:

      Centered Text
Clean Version for Your Notes

Font properties

font-size → controls the size of text

font-weight → controls how bold text is

font-style → controls italic or normal text

font-variant → controls special text styles like small caps

Text formatting

text-transform → changes capitalization

text-indent → adds space at the start of the first line

Spacing

letter-spacing → space between letters

word-spacing → space between words

Alignment

text-align → controls how text is aligned (left, right, center)
